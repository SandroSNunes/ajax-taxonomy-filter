How to define the content that will be reloaded by ajax?

You need to set the HTML element that will contain the content that will be reloaded by ajax.
You can set that in:

WordPress Admin > Settings > Ajax Texonomy Filter > HTML content element

You can add the HTML element, id or class. Ex: main, #primary, .products, ...

The pagination doesn't work?

No, as requested, the pagination is made using lazy load, so you only need to scroll down to see the next page.
