Ajax Taxonomy Filter

Description FAQ Changelog

Filter archive lists by taxonomy.

Shortcodes

Show the taxonomy filter.

[ajax-taxonomy-filter taxonomy="categories"]

All shortcode attributes.

[ajax-taxnomomy-filter taxonomy-name="" placeholder="" orderby="" order="" multi_select="0" narrow="0" show_count="0" hide_empty="0" show_hierarchy="0" ]

$args = [
    'taxonomy'       => '',
    'placeholder'    => '',
    'orderby'        => '',
    'order'          => '',
    'multi_select'   => false,
    'narrow'         => false,
    'show_count'     => false,
    'hide_empty'     => false,
    'show_hierarchy' => false,
];
